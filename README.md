# movie-site-keyword-study
本项目主要用于分析影视类网站的关键词布局策略与标题组合模式，通过对首页及内页的标题写法、描述结构及标签使用方式进行归档整理，为SEO页面优化提供参考样本。仓库不包含影视资源或在线播放内容，仅限于公开页面信息记录。所有资料整理自公开网络环境，仅供研究与结构对比参考，不构成推荐或商业宣传。
